팝업 열기

팝업을 열어서 트리 구조와 폴더 아이콘이 제대로 작동하는지 테스트해보세요.

디버깅 도구

팝업이 열린 후 브라우저 개발자 도구 Console에서 다음 명령어를 실행하세요:

  • debugTreeStructure() - 트리 구조 상세 분석
  • testFirstTwisty() - 첫 번째 트위스티 클릭 테스트
  • showAllChildUls() - 모든 하위 카테고리 표시
  • forceExpandAll() - 모든 하위 카테고리 강제 열기
